* Aufgabe 1
** a
\sum(k=1, n, k^2) = n(n + 1)(2n + 1) / 6

IA:

  Es gilt

    \sum(k=1, 1, k^2) = 1(1 + 1)(2*1 + 1) / 6

                     = 1 * 2 * 3 / 6

                     = 1

                     = 1^2


IV:

  Es gelte für ein beliebiges n aus N mit n > 0

    \sum(k=1, n, k^2) = n(n + 1)(2n + 1) / 6

IB:

  Dann gilt

    \sum(k=1, n + 1, k^2) = (n + 1)(n + 2)(2n + 3) / 6


IS:

  Es gilt

    \sum(k=1, n + 1, k^2) = \sum(k=1, n, k^2) + (n + 1)^2

                         = n(n + 1)(2n + 1) / 6 + (n + 1)^2

                         = n(n + 1)(2n + 1) / 6 + 6(n + 1)^2 / 6

                         = ( n(n + 1)(2n + 1) + 6(n + 1)^2 ) / 6

                         = ( n(n + 1)(2n + 1) + 6(n + 1)(n + 1) ) / 6

                         = ( n(n + 1)(2n + 1) + 3(n + 1)(2n + 2) ) / 6

                         = ( n(n + 1)(2n + 1) + 3(n + 1)(2n + 1) + 3(n + 1) ) / 6

                         = ( (n + 2)(n + 1)(2n + 1) + (n + 1)(2n + 1) + 3(n + 1) ) / 6

                         = ( (n + 2)(n + 1)(2n + 1) + (n + 1)(2n + 4) ) / 6

                         = ( (n + 2)(n + 1)(2n + 1) + 2(n + 1)(n + 2) ) / 6

                         = (n + 2)(n + 1)(2n + 3) / 6


Damit gilt für alle n aus N mit n > 0

  \sum(k=1, n, k^2) = n(n + 1)(2n + 1) / 6

** b
\prod(k=2, n, 1 - 1/k^2) = (n + 1) / 2n

IA:

  Es gilt

    \prod(k=2, 2, 1 - 1/k^2) = (2 + 1) / 4

                            = 3/4

                            = 1 - 1/4


IV:

  Es gelte für ein beliebiges n aus N mit n > 1

    \prod(k=2, n, 1 - 1/k^2) = (n + 1) / 2n


IB:

  Dann gilt

    \prod(k=2, n + 1, 1 - 1/k^2) = (n + 2) / 2(n + 1)

IS:

  Es gilt

    \prod(k=2, n + 1, 1 - 1/k^2) = \prod(k=2, n, 1 - 1/k^2) * (1 - 1/(n + 1)^2)

                                = (n + 1) / 2n * (1 - 1/(n + 1)^2)

                                = (n + 1) / 2n * ((n+1)^2 / (n+1)^2 - 1/(n + 1)^2)

                                = (n + 1) / 2n * ( ((n+1)^2 - 1) / (n+1)^2 )

                                = (n + 1) / 2n * ( ((n+1) - 1)((n+1) + 1) / (n+1)^2 )

                                = (n + 1) / 2n * ( n(n + 2) / (n+1)(n+1) )

                                = (n + 1) * n(n + 2) / 2n(n + 1)(n + 1)

                                = (n + 2) / 2(n + 1)


Damit gilt für alle n aus N mit n > 1

  \prod(k=2, n, 1 - 1/k^2) = (n + 1) / 2n

\Leftrightarrow 16  \ge 16

  Es gelte für ein beliebiges n aus N mit n \ge 4

    2^n \ge n^2


IB:

  Dann gilt

    2^(n + 1) \ge (n + 1)^2


IS:

  Es gilt

    2^(n + 1) \ge (n + 1)^2

\Leftrightarrow 2 * 2^n   \ge 2 * n^2 = n^2 + 2n + 1

\Rightarrow n^2 = 2n + 1

\Leftrightarrow 0 = n^2 - 2n - 1

\Rightarrow n1 = 1 - sqrt(2) und n2 = 1 - sqrt(2)

\Rightarrow n1 < 4 und n2 < 4

\Rightarrow n^2 > 2n + 1

\Rightarrow 2 * n^2 > n^2 + 2n + 1 = (n + 1)^2

\Rightarrow 2 * 2^n > (n + 1)^2

\Leftrightarrow 2^(n + 1) > (n + 1)^2


Damit gilt für alle n aus N mit n \ge 4

  2^n \ge n^2
