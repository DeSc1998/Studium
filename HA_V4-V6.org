IA:

  Es gilt

    sun(k=1, 1, k^2) = 1(1 + 1)(2*1 + 1) / 6

                     = 1 * 2 * 3 / 6

                     = 1

                     = 1^2


IV:

  Es gelte für ein beliebiges n aus N mit n > 0

    sun(k=1, n, k^2) = n(n + 1)(2n + 1) / 6

IB:

  Dann gilt

    sun(k=1, n + 1, k^2) = (n + 1)(n + 2)(2n + 3) / 6


IS:

  Es gilt

    sun(k=1, n + 1, k^2) = sun(k=1, n, k^2) + (n + 1)^2

                         = n(n + 1)(2n + 1) / 6 + (n + 1)^2

                         = n(n + 1)(2n + 1) / 6 + 6(n + 1)^2 / 6

                         = ( n(n + 1)(2n + 1) + 6(n + 1)^2 ) / 6

                         = ( n(n + 1)(2n + 1) + 6(n + 1)(n + 1) ) / 6

                         = ( n(n + 1)(2n + 1) + 3(n + 1)(2n + 2) ) / 6

                         = ( n(n + 1)(2n + 1) + 3(n + 1)(2n + 1) + 3(n + 1) ) / 6

                         = ( (n + 2)(n + 1)(2n + 1) + (n + 1)(2n + 1) + 3(n + 1) ) / 6

                         = ( (n + 2)(n + 1)(2n + 1) + (n + 1)(2n + 4) ) / 6

                         = ( (n + 2)(n + 1)(2n + 1) + 2(n + 1)(n + 2) ) / 6

                         = (n + 2)(n + 1)(2n + 3) / 6


Damit gilt für alle n aus N mit n > 0

  sun(k=1, n, k^2) = n(n + 1)(2n + 1) / 6

prod(k=2, n, 1 - 1/k^2) = (n + 1) / 2n

IA:

  Es gilt

    prod(k=2, 2, 1 - 1/k^2) = (2 + 1) / 4

                            = 3/4

                            = 1 - 1/4


IV:

  Es gelte für ein beliebiges n aus N mit n > 1

    prod(k=2, n, 1 - 1/k^2) = (n + 1) / 2n


IB:

  Dann gilt

    prod(k=2, n + 1, 1 - 1/k^2) = (n + 2) / 2(n + 1)

IS:

  Es gilt

    prod(k=2, n + 1, 1 - 1/k^2) = prod(k=2, n, 1 - 1/k^2) * (1 - 1/(n + 1)^2)

                                = (n + 1) / 2n * (1 - 1/(n + 1)^2)

                                = (n + 1) / 2n * ((n+1)^2 / (n+1)^2 - 1/(n + 1)^2)

                                = (n + 1) / 2n * ( ((n+1)^2 - 1) / (n+1)^2 )

                                = (n + 1) / 2n * ( ((n+1) - 1)((n+1) + 1) / (n+1)^2 )

                                = (n + 1) / 2n * ( n(n + 2) / (n+1)(n+1) )

                                = (n + 1) * n(n + 2) / 2n(n + 1)(n + 1)

                                = (n + 2) / 2(n + 1)


Damit gilt für alle n aus N mit n > 1

  prod(k=2, n, 1 - 1/k^2) = (n + 1) / 2n

    2^n >= n^2


IB:

  Dann gilt

    2^(n + 1) >= (n + 1)^2

  Es gilt

    2^(n + 1) >= (n + 1)^2

<=> 2 * 2^n   >= 2 * n^2 = n^2 + 2n + 1

 => n^2 = 2n + 1

<=> 0 = n^2 - 2n - 1

 => n1 = 1 - sqrt(2) und n2 = 1 - sqrt(2)

 => n1 < 4 und n2 < 4

 => n^2 > 2n + 1

 => 2 * n^2 > n^2 + 2n + 1 = (n + 1)^2

 => 2 * 2^n > (n + 1)^2

<=> 2^(n + 1) > (n + 1)^2


Damit gilt für alle n aus N mit n >= 4

  2^n >= n^2
